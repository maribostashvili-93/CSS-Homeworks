# CSS Homeworks

თითოეული დავალება მოთავსებულია ცალკე საქაღალდეში და შეიცავს საკუთარ `README.md` ფაილს, სადაც წერია:
- დავალების აღწერა
- ტექნიკური მოთხოვნები
- Figma link
- acceptance criteria

## როგორ უნდა იმუშაონ სტუდენტებმა

1. გახსენით სასურველი დავალების საქაღალდე.
2. წაიკითხეთ იმ საქაღალდეში არსებული `README.md`.
3. გახსენით მითითებული Figma link.
4. დაიწყეთ მუშაობა თქვენს რეპოზიტორიაში.
5. შექმენით მინიმუმ:
   - `index.html`
   - CSS ფაილი (მაგალითად `style.css`)
